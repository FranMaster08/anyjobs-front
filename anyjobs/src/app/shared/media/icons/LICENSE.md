# Iconos del slider de medios (AnyJobs)

Los archivos SVG de esta carpeta (`slider-*.svg`) son obra de AnyJobs para uso exclusivo en el producto AnyJobs.

`slider-like.svg` / `slider-like-active.svg`: rayo para like.

Copia servida en runtime: `public/media-slider-icons/` (rutas absolutas `/media-slider-icons/...` en estilos).

Se permiten modificar y redistribuir únicamente como parte de este repositorio y despliegues del producto.

No están sujetos a licencias de iconografía de terceros (emojis del sistema, TikTok, Instagram, etc.).
